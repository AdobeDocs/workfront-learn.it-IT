# Configurare le impostazioni globali della bozza

Avvia le configurazioni di sistema per la verifica delle bozze in [!DNL Workfront] stabilendo impostazioni predefinite globali che influiscono sugli utenti. Imposta queste impostazioni predefinite affinché gli utenti creati dispongano delle impostazioni minime necessarie per iniziare a utilizzare le funzionalità di verifica delle bozze. Questo consente inoltre di risparmiare tempo mentre si lavora con le impostazioni per la verifica delle bozze.

* Assegnare ruoli bozza predefiniti
* Stabilire le impostazioni account predefinite
* Impostare le impostazioni di bozza predefinite

Se hai già iniziato a creare utenti prima di stabilire queste impostazioni predefinite, non preoccuparti. Puoi modificare tutte queste informazioni sui singoli profili utente.
